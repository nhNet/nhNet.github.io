Online Websites

Some libraries (e.g. the Google SignIn one) require your wesite to be online and not a local website. A local website is an html file on your computer. My browser if fully online, thanks to Github Pages. Github is a place where developers can store their files and let other users collaborate on the same file at the same time. It also has a web hosting service that can be activated by going into your repository's Settings. Then, scroll down to the Github Pages section. You can create a new Branch called gh-pages or you can select the master branch as the source.

What Github Pages does is it takes your files and makes them available online (.html files as websites).
A good thing Github Pages does is so you don't have to type in the "index.html" part e.g.
        1. https://nhnet.github.io/index.html
        2. https://nhnet.github.io
They are both the same website!
